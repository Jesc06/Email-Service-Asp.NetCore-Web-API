# Email-Service-Asp.NetCore-Web-API
*Email service configuration in web api*

---

1. Create application.json EmailSettings

   ```json
    "EmailSettings": {
      "SmtpServer": "smtp.gmail.com",
      "SmtpPort": 587,
      "SmtpUser": "your email",
      "SmtpPassword": "your smtp password"
    }
   ```
